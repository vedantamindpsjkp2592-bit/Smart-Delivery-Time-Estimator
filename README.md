Smart Delivery Time Estimator

This project is part of the Introduction to Problem Solving & Programming flipped course.
The application estimates the time required for a parcel to reach its destination based on distance, transport mode, and traffic level.

---

1. Problem Statement
In logistics, customers often want to know how long a package will take to be delivered. Delivery time changes depending on vehicle speed and traffic conditions.
This project tries to give a simple and quick estimate using basic programming logic.

---

2. Features
User enters distance in kilometers
User selects type of transport (Bike, Car, Truck)
User selects traffic condition
Program calculates delivery duration
Displays the complete breakdown of timing

---

3. Technologies Used
Python (basic input/output, conditional statements, arithmetic)
